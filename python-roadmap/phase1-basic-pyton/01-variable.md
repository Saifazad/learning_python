# Variables

Variable ek container hai jisme value store hoti hai. Naya variable banane ke liye directly naam likho aur `=` se value do — Python me type define karne ki zarurat nahi.

```python
naam = "Aman"
age = 20
is_student = True

print(naam)
print(age)

age = 21          # value change kar sakte hain
print(age)
```

**Key points:**
- Variable naam letter ya underscore se start hona chahiye, number se nahi
- Case-sensitive hota hai (`Age` aur `age` alag hain)
- Spaces allowed nahi, underscore use karo (`my_name`)
- Value kabhi bhi change ho sakti hai
  
**Practice Questions:**
1. Apna naam, age, aur city — teen alag variables me store karo aur print karo.
2. Do numbers ko variables me store karo aur unka sum ek naye variable me store karke print karo.
3. Ek variable `price = 100` banao, phir usi variable ki value change karke `150` kar do aur dono values print karo.
4. Multiple assignment try karo: `a, b, c = 10, 20, 30` ek hi line me likho aur teeno print karo.
5. Ek variable banao jisme favourite subject store ho, aur usi variable ko use karke ek sentence print karo.
